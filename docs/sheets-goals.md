# Google Sheets Schema — Goals Tab

This tab stores **savings goals** for Phase B of the Goals update.

## Tab name
`Goals` (case-sensitive — must match exactly)

## Required columns (A–I)

| Col | Header | Type | Notes |
|---|---|---|---|
| A | `ID` | number | Auto-generated by the API using `Date.now()`. Must be unique. Don't change. |
| B | `Nama` | string | Display name, e.g. `Dana Darurat`, `Liburan Bali`. |
| C | `Target` | number | Goal amount as a plain integer (no `Rp`, no thousand separators, no decimal). E.g. `50000000`. |
| D | `Deadline` | string | Target completion date. Accepted formats: `YYYY-MM` (e.g. `2026-12`) or `YYYY-MM-DD` (e.g. `2026-12-31`). |
| E | `Kategori` | string | The savings category this goal tracks. Must match one of `SAVINGS_CATEGORIES` exactly. **This is the auto-link key** — every `Tabungan` transaction with this category will count toward this goal. |
| F | `Icon` | string (optional) | Lucide icon name, e.g. `Shield`, `Plane`, `Home`, `Target`. Leave empty to use the default `Target`. |
| G | `Color` | string (optional) | Hex color for the progress ring, e.g. `#5b8c7a` (moss/green). Leave empty to use the default moss color. |
| H | `CreatedAt` | string | ISO date the goal was created (`YYYY-MM-DD`). Auto-generated by the API on POST. Don't change. Used to compute ETA pace. |
| I | `Status` | string | `open` (default) or `settled`. Settled goals move to the "Completed" section and show "Terealisasi" badge. |

## Behavior

- **Auto-link by category**: Progress is automatically computed as the sum of all `Tabungan` transactions where:
  - `t.category === goal.Kategori` (exact match, case-sensitive)
  - `t.date >= goal.CreatedAt` (only contributions after the goal was created)
- **ETA prediction**: "At current pace: 8 months" is calculated as `(target − progress) / (progress / daysSinceCreated) / 30`.
- **Celebration trigger**: First time progress crosses from `< 100%` to `>= 100%` → confetti + toast + haptic.

## Categories whitelist

`Kategori` must match `SAVINGS_CATEGORIES` exactly:
```
Tabungan Cash, Emas, Saham
```

## Available icons (Lucide)

Common choices: `Shield`, `Plane`, `Home`, `Car`, `Heart`, `BookOpen`, `Gift`, `Laptop`, `Camera`, `PiggyBank`, `Target`, `Wallet`, `Star`, `Sun`, `Moon`, `Flag`.

## Setup steps

1. In your spreadsheet, click **+** at the bottom to add a new sheet
2. Name it `Goals`
3. In row 1, add the headers above (one per column, A through H)
4. In row 2+, add your goal rows. **You can leave column A (`ID`) empty** — the API will auto-fill it on first POST. If you add rows manually, just use any unique number (e.g. a timestamp).
