📋 How to Generate PNG Files

Option 1: Online Converter (Easiest)

  1. Go to svgtopng.com or convertio.co
  2. Upload public/icons/icon.svg
  3. Set size to 192×192 → download as icon-192.png
  4. Set size to 512×512 → download as icon-512.png
  5. Place both files in public/icons/

Option 2: Browser Screenshot

  1. Right-click each icon above → "Save image as..." (may save as SVG)
  2. Or use browser DevTools → Screenshot node

Option 3: Figma / Canva

  1. Import the SVG into Figma or Canva
  2. Export as PNG at 192×192 and 512×512
